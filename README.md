# NEXORA SYSTEMS - Landing Page

Transformación digital y automatización inteligente para empresas.

## 🚀 Descripción

Landing page profesional para NEXORA SYSTEMS, desarrollada con Next.js 14, TailwindCSS y Framer Motion. Diseñada para convertir visitantes en clientes potenciales a través de una experiencia premium y optimizada.

## ✨ Características

- **Next.js 14** con App Router
- **TailwindCSS** para diseño personalizado
- **Framer Motion** para animaciones fluidas
- **Dark Mode** premium
- **Responsive Design** mobile-first
- **SEO Optimizado** con metadata completa
- **Componentes modulares** y reutilizables
- **Formulario funcional** con validación
- **Animaciones avanzadas** y micro-interacciones

## 📁 Estructura del Proyecto

```
├── app/
│   ├── globals.css          # Estilos globales
│   ├── layout.tsx           # Layout principal
│   └── page.tsx            # Página principal
├── components/
│   ├── Hero.tsx             # Sección hero con animaciones
│   ├── ProblemaReal.tsx     # Problema que resuelve
│   ├── Resultados.tsx        # Resultados y beneficios
│   ├── Metodologia.tsx       # Proceso de trabajo
│   ├── Impacto.tsx           # Métricas y resultados
│   ├── LeadMagnet.tsx        # Formulario de captación
│   ├── Calendario.tsx         # Agenda de citas
│   ├── FAQ.tsx               # Preguntas frecuentes
│   ├── Blog.tsx               # Blog estratégico
│   ├── CTAFinal.tsx           # Llamada a acción final
│   └── Footer.tsx             # Footer profesional
├── package.json
├── tailwind.config.js
├── tsconfig.json
├── next.config.js
└── postcss.config.js
```

## 🛠️ Instalación

1. Clonar el repositorio:
```bash
git clone <repository-url>
cd nexora-systems
```

2. Instalar dependencias:
```bash
npm install
```

3. Ejecutar en desarrollo:
```bash
npm run dev
```

4. Construir para producción:
```bash
npm run build
```

## 🚀 Despliegue

### Netlify
1. Conectar repositorio a Netlify
2. Configurar build command: `npm run build`
3. Configurar publish directory: `.next`
4. Deploy automático en cada push

### Hostinger
1. Construir proyecto: `npm run build`
2. Subir carpeta `.next` al servidor
3. Configurar Node.js y variables de entorno
4. Configurar dominio y SSL
